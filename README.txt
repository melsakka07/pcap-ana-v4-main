SIP Message Analyzer
===================

Prerequisites:
- Wireshark/TShark must be installed
- Windows operating system

Usage:
1. Double-click SIP-Analyzer.exe
2. Select the folder containing your PCAP files when prompted
3. Select where you want the analysis results to be saved
4. Wait for the analysis to complete
5. Check the output folder for results
